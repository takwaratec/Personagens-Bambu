# 🎬 Google Flow — Passo a Passo

## URL

[https://labs.google/fx/pt/tools/flow/](https://labs.google/fx/pt/tools/flow/)

## Fluxo Completo

### 1. Crie um projeto
- Clique em **Novo Projeto** e dê um nome

### 2. Crie um personagem
- Vá na aba **Personagens**
- Arraste a **Ficha de Consistência** como ingrediente
- Descreva o personagem em linguagem natural
- Selecione e personalize a **voz**
- Salve o personagem

### 3. Crie o vídeo

1. Mude para modo **"Elementos"**
2. Escolha o formato (16:9, 9:16, 1:1)
3. Selecione modelo (**Omni Flash** = melhor custo-benefício)
4. **Desative o Agente** (causa erros)
5. Use **@Personagem** no prompt (arroba + nome)
6. Traduza o prompt para inglês, **mantendo o diálogo em português**
7. Defina duração dos shots
8. Gere em **720p** (depois faça upscale)

### Erros comuns

- ❌ Esquecer de marcar **@Personagem**
- ❌ Deixar o **Agente ativado**
- ❌ Traduzir o **diálogo** junto com o prompt
- ❌ Editar dentro do Flow (use Cap Cut)
