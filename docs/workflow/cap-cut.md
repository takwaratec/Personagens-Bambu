# ✂️ Cap Cut — Pós-Produção

## Fluxo pós-geração no Flow

1. **Monte os clipes** na timeline em ordem
2. **Extraia vocais**: clique no clipe → Áudio separado → Vocais
3. **Troque voz**: use "Mudar voz" para selecionar novo estilo
4. **Remova marca d'água**: ferramenta "Remoção de IA" → lápis → selecionar → remover
5. **Adicione música**: aba Áudio → Músicas → pesquise por estilo
6. **Ajuste volume** da música: -8 a -11 dB (não abafar a voz)
7. **Adicione transições, textos e efeitos**
8. **Exporte** na resolução desejada
