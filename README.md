# 🎋 Personagens Bambu

Repositório para construção, documentação e backup dos personagens da **Mentoria Tecnologia Takwara**, baseados nos **7 Lições do Bambu** e nos **7 Pilares de Edgar Morin**.

## Estrutura

```
Personagens-Bambu/
├── personagens/            # Fichas de cada personagem
│   └── template.md         # Template de ficha de personagem
├── prompts/                # Prompts para geração (Google Flow, ChatGPT, Gemini)
├── assets/                 # Imagens, vídeos, referências visuais
├── design-system/          # Design system da marca
└── flow-projetos/          # Links e exports dos projetos Google Flow
```
