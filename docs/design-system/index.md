# 🎨 Design System — Personagens Bambu

## Estrutura dos 7 Personagens

Cada personagem incorpora **um aspecto do uso do bambu** — para o bem ou para o mal.  
O DNA visual de cada um reflete sua função no ecossistema de conhecimento.

| # | Personagem | Função | Alinhamento | Biotipo |
|---|---|---|---|---|
| 🌟 | **Bambusa** | Âncora / Anfitriã | Neutro | Flexível, dançante, acolhedora |
| 🎙️ | **Vulgaris** | Coadjuvante / Entrevistador | Neutro | Curioso, inquieto, em movimento |
| 🔬 | **Sabia-mamona** | Química / Tratamento | Ambos | Anciã, curvada, lupa, frascos |
| 🌿 | **Cacique-Asper** | Saberes indígenas | Ambos | Guerreiro, penas, base larga |
| 🏹 | **Taboca** | Extrativismo / Comunidades | Ambos | Encorpado, nós marcados, chapéu |
| 📊 | **Dr. Burocrata** | Monocultura / Lucro | ❌ Mau | Terno de jornal, pasta, óculos |
| 🌱 | **Tuldoides** | Cooperativas / Organização social | Ambos | Esguio, prancheta, articulador |
| ⚙️ | **Dr. Motosserra** | Negacionismo ético | ❌ Mau | Rachado, motosserra, dissimulado |

## Definição do DNA Visual

### 1. Paleta de cores base (Terra + Bambu)

| Cor | Hex | Uso |
|---|---|---|
| Verde Bambu Escuro | `#2D5016` | Base dos personagens, contornos |
| Verde Bambu Claro | `#6B8F3A` | Vestuário principal, elementos naturais |
| Terra Queimada | `#8B5E3C` | Peles, texturas orgânicas |
| Areia | `#D4C4A8` | Fundos, elementos secundários |
| Céu Amanhecer | `#E8D5B7` | Iluminação quente, planos de fundo |
| Rio Profundo | `#2C5F7C` | Detalhes, olhos, água |

### 2. Biotipos dos personagens

Cada personagem tem um perfil corporal distinto:

| Personagem | Biotipo | Silhueta | Postura |
|---|---|---|---|
| 1 — Resiliência | Flexível, esguio | Curvas suaves | Inclinado, nunca quebrado |
| 2 — Etapas | Robusto, encorpado | Retangular com divisórias | Ereto, firme |
| 3 — Fundação | Largo, base sólida | Triangular (base larga) | Agachado, raízes visíveis |
| 4 — Visão | Alto, magro, alongado | Linha vertical ascendente | Esticado, cabeça erguida |
| 5 — Humildade | Médio, aparência oca | Circular, centro vazio | Despretensioso, leve |
| 6 — Coletivo | Conectado, ramificado | Vários braços/galhos | Entrelaçado |
| 7 — Ritmo | Assimétrico, único | Desigual, surpreendente | Fora do eixo, charmoso |

### 3. Estilo visual

- **Técnica:** Studio photography, hiper-realista com toque estilizado
- **Iluminação:** Quente (golden hour), suave, com sombras orgânicas
- **Fundo:** Neutro com gradiente terra-céu, ou ambiente natural
- **Textura:** Pele com textura vegetal sutil (fibras de bambu)
- **Expressão:** Neutra na ficha, variável nas cenas

## Fluxo no Stitch (Google)

1. Acesse https://stitch.withgoogle.com/
2. Crie um novo **Design System**
3. Alimente:
   - A paleta de cores acima
   - As silhuetas de cada personagem
   - As diretrizes de iluminação e fundo
4. Exporte o Design System como referência
5. Use no Claude Design para refinar tipografia e aplicações
6. Salve o resultado em `design-system/` do repositório

> 💡 O Design System no Stitch vira um arquivo de referência que você anexa **junto com a Character Consistency Sheet** em cada geração no Google Flow, garantindo consistência cromática e estilística entre todos os personagens.
