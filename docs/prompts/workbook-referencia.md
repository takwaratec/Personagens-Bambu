## Design System (o DNA da marca)

**Ferramentas:** Stitch (Google) + Claude Design

**Prompt básico:**
```
Meu negócio é [DESCREVA] com a marca [NOME].

Quero criar um Design System para ela, para depois a partir desse Design System,
criar um site e materiais para divulgação nas redes sociais. Em anexo, coloquei
um exemplo de uma imagem com uma coloração que acho bacana usar.
```

> 💡 Sempre adicione uma referência visual (imagem ou URL de inspiração).

## Logotipos

**Ferramentas:** ChatGPT, Gemini, Google Flow

**Prompt básico:**
```
Em anexo o design system da minha marca chamada [NOME].

Gere para mim 4 versões de logos estilo [ESTILO], lembrando que meu nicho é [NICHO]
```

**Estilos para testar:** Minimalista (Apple, Nike), Vintage (Levi's), Clássico (Rolex), Divertido (Nintendo), Brutalista (CAT)

**Recomendações:**
- Gere uma versão fundo escuro, uma claro, uma transparente
- Edite iterativamente no ChatGPT ou leve ao Canva (Camadas Mágicas)

## Rebranding

**Prompt completo para rebranding de logotipo existente:**

```
Você é um diretor criativo especializado em branding, identidade visual e design
de logotipos modernos.

Sua tarefa é realizar um rebranding completo de uma logo existente, utilizando
como base:
1. A logo atual (anexada)
2. Uma ou mais imagens de referência (anexadas)

[Segue estrutura completa com 4 versões: Evolução direta, Minimalista premium,
Criativa/diferenciada, Comercial/versátil]
```

**Diretrizes:** Fundo limpo, alta definição, composição centralizada, sem mockups, sem efeitos exagerados, sem 3D desnecessário.

## Stories

**Ferramentas:** ChatGPT, Gemini, Google Flow

**Prompt:**
```
Gere para mim uma imagem que será um post em formato stories 9:16 de Instagram
para minha empresa (logo em anexo) usando o design system em anexo também.
A ideia do post é ser um teaser, com alguma frase do tipo "Aguarde..." para
que a pessoa fique na expectativa de algo novo que vai surgir
```

> 💡 Anexe sempre o design system + logotipo quando necessário.

## Imagem com Pessoas / Ficha de Consistência do Personagem

**Ferramentas:** ChatGPT, Gemini, Google Flow

**Prompt para "Character Consistency Sheet":**

```
Analise cuidadosamente todas as imagens anexadas.

Sua tarefa é criar uma Character Consistency Sheet profissional seguindo
o mesmo padrão visual da ficha de referência.

[Preservação da identidade: estrutura facial, olhos, nariz, boca, cabelo,
biotipo, idade, tom de pele, características únicas]

Layout:
- Parte superior: Retrato frontal em close-up
- Parte inferior: Perfil esquerdo, Perfil direito
- Área principal: Corpo inteiro frontal, Corpo inteiro traseiro

Direção: Hiper-realista, estúdio profissional, fundo cinza neutro,
iluminação uniforme, expressão neutra, nitidez máxima.
```

> 💡 Anexe: close frontal, vistas laterais, corpo inteiro, vista traseira.
> Use a ficha como referência permanente para todas as gerações futuras.

## Anúncio Estático

**Estrutura de prompt completa:**

```
OBJETIVO: [Objetivo da campanha]
PRODUTO: [Descrição da oferta]
PÚBLICO: [Quem verá o anúncio]
CONCEITO: [Ideia criativa central]
HEADLINE: [Texto principal]
SUBHEADLINE: [Texto complementar]
CTA: [Chamada para ação]
DIREÇÃO DE ARTE: [Estilo visual]
ELEMENTOS VISUAIS: [O que deve aparecer]
COMPOSIÇÃO: [Organização dos elementos]
IDENTIDADE VISUAL: [Design system + referências em anexo]
FORMATO: [4:5, 9:16, etc.]
```

> 💡 Use o GPT personalizado para versão automatizada.

## Carrossel

**Ferramentas:** ChatGPT, Gemini, Google Flow

**Abordagem recomendada:** Gere slide a slide, iterando cada um.

**Exemplo slide 1:**
```
Estou criando um carrossel no formato 4:5 para divulgar [SERVIÇO/PRODUTO].
Em anexo, o logotipo, Design System e uma foto minha.

Vamos começar gerando a imagem do slide 1.
Nele você deve colocar a headline: "[TEXTO]"
E como imagem chamativa: [DESCRIÇÃO DA CENA]
```

> 💡 Gerar individualmente e iterar ao longo do caminho funciona melhor que gerar tudo de uma vez.

## Vídeo de Animação

**Ferramentas:** Google Flow, Kling AI, Magnific

**Prompt exemplo:**
```
The character appears walking across a soccer field with a ball under his arm,
saying in Brazilian Portuguese: "Olá, amiguinhos! Meu nome é [NOME] e eu estou
aqui para divertir vocês. Bora bater uma bolinha?"
```

**Multi-shot:**
```
SHOT 1: [Descrição da ação]
SHOT 2: [Descrição da ação]
SHOT 3: [Descrição da ação]
```

> 💡 Use SEMPRE a Ficha de Consistência do Personagem na criação de vídeos.

## Vídeo de Avatar

**Ferramentas:** Google Flow, Kling AI, Magnific

**Prompt multi-shot (com marcação de ingredientes):**
```
- @img1 [descrição da cena e ação, duração em segundos]
- @img2 [descrição da cena e ação, duração em segundos]
```

> 💡 Especifique segundos para cada shot. Marque ingredientes anexados com @ no prompt.

## Clones

**Ferramentas:** HeyGen, Google Flow

**Dicas:**
1. Quanto melhor o vídeo de referência, melhor o clone
2. Prefira criar clone a partir de ÁUDIO ao invés de texto
3. Use Google Flow para criar ferramentas customizadas (também chamadas de "clones")

## Google Flow — Visão Geral

**URL:** https://labs.google/fx/pt/tools/flow/
**O que é:** Estúdio criativo de IA do Google Labs

### Capacidades

| Funcionalidade | Descrição |
|---|---|
| **Imagens (Imagen)** | Geração texto→imagem, edição precisa, consistência de assunto |
| **Vídeos (Veo)** | Vídeos cinematográficos com áudio nativo, física realista |
| **Ferramentas/Clones** | Crie ferramentas personalizadas em linguagem natural |
| **Agente (Gemini)** | Assistente criativo integrado ao projeto |
| **Áudio (Lyria)** | Música e áudio de alta fidelidade |

### Funcionalidades
- Texto animado em vídeos
- Redimensionamento (qualquer proporção)
- Filtros visuais (lo-fi, glitch, vintage)
- Editor de camadas
- Storyboard: roteiro → elenco → visualização
- Vídeos beat-driven (ritmo musical)
- Captura 360° a partir de imagem
- Grade de imagens
- Efeitos de pós-produção
- Composição de ambiente
