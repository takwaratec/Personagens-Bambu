# 🎋 Personagens Bambu

Bem-vindo ao repositório de construção dos **Personagens Bambu** — personagens baseados nas **7 Lições do Bambu** e nos **7 Pilares de Edgar Morin**, criados com IA generativa (Google Flow, Imagen, Veo, ChatGPT e Gemini).

## Objetivo

Construir um elenco de personagens que materializem os princípios filosóficos da **Mentoria Tecnologia Takwara**, usando ferramentas de IA para manter consistência visual, narrativa e vocal.

## Estrutura do site

| Seção | Descrição |
|---|---|
| [Personagens](personagens/template.md) | Fichas individuais de cada personagem |
| [Workflow](workflow/google-flow.md) | Passo a passo de criação com Google Flow |

| [Projetos Flow](flow-projetos/index.md) | Links para projetos no Google Labs |
| [Design System](design-system/index.md) | Diretrizes visuais da marca |

## Ferramentas

| Ferramenta | Uso |
|---|---|
| **Google Flow** ([labs.google/fx](https://labs.google/fx/pt/tools/flow/)) | Estúdio criativo — imagens, vídeos, clones |
| **ChatGPT** | Logotipos, variações de imagem |
| **Gemini** | Imagens, storyboards |
| **HeyGen** | Clones de avatar para clientes |
| **Cap Cut** | Edição e pós-produção de vídeos |
