### Instruções de geração no Flow

1. Crie um projeto **"Personagens-Bambu-Trail-30s"**
2. Para cada clip:
   - Ative o personagem com **@NomeDoPersonagem**
   - Traduza o prompt para inglês, **mantendo o diálogo em português entre aspas**
   - Use **Omni Flash** (modelo), formato **16:9** (horizontal) ou **9:16** (vertical)
   - Duração: conforme tabela acima
   - Resolução: **720p** (depois faça upscale)
   - **Desative o Agente**
3. Após gerar todos os 7 clips, baixe cada um
4. No **Cap Cut**: monte na ordem, ajuste transições, adicione música de fundo, sincronize

### Pós-produção (Cap Cut)
- Música: pesquisa por "bambu" ou "world percussion" na biblioteca do Cap Cut
- Volume da música: -10 dB
- Transições: fade cruzado de 0.5s entre clips
- Texto final: overlay "Personagens Bambu — Tecnologia Takwara" nos últimos 2s
